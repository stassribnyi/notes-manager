# Simple notes manager application written with node.js

## Usage

- Add note
  ```sh
  node app.js add --title="Shopping list" --body="Buy some milk and cornflakes"
  ```
- List all notes
  ```sh
  node app.js list
  ```
- Remove note
  ```sh
  node app.js read --title="Dry cleaning"
  ```
- Remove note
  ```sh
  node app.js remove --title="Dry cleaning"
  ```
